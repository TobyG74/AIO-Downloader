# AIO Downloader v1.0.2 Release Notes

## [ EN ]
### Update v1.0.2 - New Platforms & Improvements
- **New Platforms:**
  - **BiliBili TV** - Download videos from BiliBili TV with automatic audio merging
    * Support for multiple URL formats (bili.im, b23.tv, bilibili.tv)
    * Video thumbnail and title extraction from video pages
    * Multiple quality selection with H.264 codec filtering
  - **Douyin (抖音)** - Download videos from Chinese version of TikTok
    * Quality selection for videos
    * Support for Douyin share links
  - **SoundCloud** - Download tracks with metadata
    * High-quality audio download
    * Track metadata and artwork extraction

- **Improvements:**
  - Enhanced video download with proper CDN headers (fixes 403 errors)
  - History now displays video thumbnails for better organization
  - Platform list reorganized with Douyin placed below TikTok
  - Replaced discontinued ffmpeg package with maintained alternative
  - Dedicated result widgets for all platforms

- **Technical:**
  - Added support for short URL redirects (bili.im, b23.tv)
  - Implemented JSON-LD metadata extraction for thumbnails
  - FFmpeg integration for video/audio merging
  - AGP 9.0+ compatibility fix

- **Platform Baru:**

# AIO Downloader v1.0.1 Release Notes

### Update v1.0.1 - Enhanced Features
- **New Features:**
  - Threads support - Download videos and images from Threads by Instagram
  - WebView verification for all platforms - Improved cookie handling
  - Enhanced history UI with platform icons and media thumbnails
  - Contributors section in About dialog

- **Improvements:**
  - Extended WebView cookie support to Instagram, Facebook, Twitter, Threads, YouTube, and Spotify
  - Better thumbnail display for downloaded media
  - Improved platform detection and URL parsing
  - Enhanced localization for all supported languages

- **Bug Fixes:**
  - Fixed WhatsApp Status screen initialization error
