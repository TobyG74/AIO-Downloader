# AIO Downloader v1.0.2 Release Notes

## [ EN ]
### Update v1.0.2 - Bilibili Support & Improvements
- **New Features:**
  - BiliBili (Bstation) downloader - Download videos from BiliBili TV with audio merging
  - Support for multiple BiliBili URL formats (bili.im, b23.tv, bilibili.tv)
  - Video thumbnail and title extraction from video pages
  - Automatic video and audio merging using FFmpeg

- **Improvements:**
  - Enhanced Bilibili download with proper CDN headers (fixes 403 errors)
  - History now displays video thumbnails for better organization
  - Platform list reorganized with Douyin placed below TikTok
  - Replaced discontinued ffmpeg package with maintained alternative

- **Technical:**
  - Added support for short URL redirects (bili.im, b23.tv)
  - Implemented JSON-LD metadata extraction for thumbnails
  - Quality selection with H.264 codec filtering

---

## [ ID ]
### Pembaruan v1.0.2 - Dukungan Bilibili & Peningkatan
- **Fitur Baru:**
  - Downloader BiliBili (Bstation) - Unduh video dari BiliBili TV dengan penggabungan audio
  - Dukungan untuk berbagai format URL BiliBili (bili.im, b23.tv, bilibili.tv)
  - Ekstraksi thumbnail dan judul video dari halaman video
  - Penggabungan video dan audio otomatis menggunakan FFmpeg

- **Peningkatan:**
  - Download Bilibili yang ditingkatkan dengan header CDN yang tepat (memperbaiki error 403)
  - Riwayat kini menampilkan thumbnail video untuk organisasi yang lebih baik
  - Daftar platform direorganisasi dengan Douyin ditempatkan di bawah TikTok
  - Mengganti paket ffmpeg yang tidak lagi dikelola dengan alternatif yang aktif

- **Teknis:**
  - Menambahkan dukungan redirect URL pendek (bili.im, b23.tv)
  - Implementasi ekstraksi metadata JSON-LD untuk thumbnail
  - Pemilihan kualitas dengan filter codec H.264

# AIO Downloader v1.0.1 Release Notes

### Update v1.0.1 - Enhanced Features
- **New Features:**
  - Threads support - Download videos and images from Threads by Instagram
  - WebView verification for all platforms - Improved cookie handling
  - Enhanced history UI with platform icons and media thumbnails
  - Contributors section in About dialog

- **Improvements:**
  - Extended WebView cookie support to Instagram, Facebook, Twitter, Threads, YouTube, and Spotify
  - Better thumbnail display for downloaded media
  - Improved platform detection and URL parsing
  - Enhanced localization for all supported languages

- **Bug Fixes:**
  - Fixed WhatsApp Status screen initialization error

  - Fixed UI overflow issues
