# AIO Downloader v1.0.1 Release Notes

## [ EN ]
### Update v1.0.1 - Enhanced Features
- **New Features:**
  - Threads support - Download videos and images from Threads by Instagram
  - WebView verification for all platforms - Improved cookie handling
  - Enhanced history UI with platform icons and media thumbnails
  - Contributors section in About dialog

- **Improvements:**
  - Extended WebView cookie support to Instagram, Facebook, Twitter, Threads, YouTube, and Spotify
  - Better thumbnail display for downloaded media
  - Improved platform detection and URL parsing
  - Enhanced localization for all supported languages

- **Bug Fixes:**
  - Fixed WhatsApp Status screen initialization error
  - Improved error handling in scrapers
  - Fixed UI overflow issues
